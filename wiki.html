Getting Started

ToSon is easy to set up. Download the files from the main repository and create your first `.toson` file.

Syntax

  • Blocks: Use curly braces { } and square brackets [ ].
  • Comments: Start with ~/<-.
  • Variables: Use let >: {...}.
  • Commands: Use say >: {...}, ask >: {...}, etc.

Commands Reference

Command Syntax Example Description
say [{Main}] {
  say >: {
    text: "Hello World"
    color: "green"
  }
}		 Print colored text
let [{Main}] {
  let >: {
    name: "x"
    value: 42
  }
}		 Assign a variable
ask [{Main}] {
  ask >: {
    have: "name"
    output: "Your name? "
  }
}		 Ask input and store it
if/else [{Main}] {
  if >: {
    if_var: "x"
    value: 42
    then: [
      say >: {
        text: "X is 42"
      }
    ]
    else: [
      say >: {
        text: "X is not 42"
      }
    ]
  }
}		 Conditional execution
repeat [{Main}] {
  repeat >: {
    times: 3
    do: [
      say >: {
        text: "Looping"
      }
    ]
  }
}		 Loop commands multiple times
choose [{Main}] {
  choose >: {
    from: ["A", "B", "C"]
    have: "choice"
  }
}		 Pick a random value from a list
set [{Main}] {
  set >: {
    name: "MainBlock"
    content: [
      let >: {
        name: "x"
        value: 10
      }
      say >: {
        text: "Block running"
      }
    ]
  }
}		 Assign variables to a block
list [{Main}] {
  list
}		 Show all implemented commands
