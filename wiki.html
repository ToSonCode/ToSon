Getting Started

ToSon is easy to set up. Download the files from the main repository and create your first `.toson` file.

Syntax

  • Blocks: Use curly braces { } and square brackets [ ].
  • Comments: Start with ~/<-.
  • Variables: let x > {"42"}.
  • Commands: say >: {...} or ask >: {...}.

Commands Reference

Command Syntax Example Description
say 
say >: {"Hello" ["green"]}
 Print colored text
let 
let x > {"42"}
 Assign variable
ask 
ask >: {have "name" output: "Your name?"}
 Ask input
if/else 
if "x" >: {value:{"42"} ... else ... }
 Conditional execution
repeat 
repeat 3 {say >:{"Hi"}}
 Loop commands multiple times
choose 
choose >: {from:["A","B"] have "choice"}
 Random selection
set 
set block "Main" >: [{key:val}]
 Assign block variables
list 
list
 Show all implemented commands
