Getting Started

ToSon is easy to set up. Download the files from the main repository and follow these steps:

  1. Include your ToSon interpreter or runtime (if applicable).
  2. Create your first `.toson` file.
  3. Write your first ToSon command (see Examples below).

Syntax

  • Blocks: Use curly braces { } and square brackets [ ] to define scopes.
  • Comments: Inline comments start with ~/<-.
  • Variables: Defined using variable: <name>.
  • Strings: Enclosed in double quotes " ".
  • Commands: Use say >: or other defined commands inside blocks.

Commands Reference

Command Description Usage
say >: Prints text or variables to output. 
say >: { 
    variable: none, 
    text: "Hello World!" 
    color: "green" 
}
variable >: Creates or assigns a variable. 
variable >: { 
    name: "userName", 
    value: "Kelsi" 
}
if >: Conditional execution based on variable or expression. 
if >: { 
    condition: userName == "Kelsi",
    do: {
        say >: { text: "Welcome!" }
    }
}

Examples

Simple command:


[{Main}]{ ~/<-this is a comment
    say >: {
        variable: none,
        text: "Hello World!"
        color: "green"
    }
}

Using variables:


[{Main}]{
    variable >: {
        name: "userName",
        value: "Kelsi"
    }
    say >: {
        variable: userName,
        text: "Welcome to ToSon!"
        color: "blue"
    }
}
