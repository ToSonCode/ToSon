About ToSon

ToSon is a minimalist, easy-to-use data format designed for developers who want a fast and intuitive way to structure data. It is lightweight, readable, and easy to integrate with modern programming environments.

Example Usage


[{Main}]{ ~/<-this is a comment
    say >: {
        variable: none,
        text: "Hello World!"
        color: "green"
    }
}
